Q2. Explain what are homogeneous coordinates and homogeneous transformation matrix and why they are useful.

Homogeneous coordinates are a coordinate system that algebraically handles points in the Euclidean plane in the same way. The transformations in homogenous systems are 4x4 matrix which describes the motion of a vector u to a vector v. This coordinate system is useful because it enables affine and projective transformations to be described as matrix manipulations in a clear and concise manner.

Q3. Explain the concept of a scenegraph and why it is useful.

A scenegraph is a strategic way of composing and visualizing complex graphics. An object that is wished to be drawned can be viewed as a tree like structure composed of other objects. much like binary trees in data structures, scenegraphs have a root node, pointing to its children (where each child can also point to other objects, and so on). The concept of a scenegraph is useful because we can draw a component of a large object and the transformations of its children are applied relative to its parent graphic. This means that we can go back to the previous transformation settings once we have finished rendering the graphics that compose a main component.
